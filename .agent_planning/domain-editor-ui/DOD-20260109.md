## Acceptance Criteria (Draft)

### AC-1: PropertyPanel Component
- [ ] PropertyPanel component exists and renders
- [ ] Accepts BlockMetadata for parameter specs
- [ ] Renders NumericInput with slider for number params
- [ ] Supports min, max, step constraints
- [ ] Fires onParamChange callback when value changes

### AC-2: Parameter Metadata System
- [ ] BlockMetadata interface defined
- [ ] GridDomain metadata with rows, cols params
- [ ] DomainN metadata with count, seed params
- [ ] Metadata registry accessible by block.type

### AC-3: Preset Support
- [ ] Preset buttons render below parameters
- [ ] GridDomain presets: 2×2, 4×4, 8×8, 16×16, 32×32
- [ ] DomainN presets: 10, 100, 1000, 5000, 10000
- [ ] Clicking preset applies all its params

### AC-4: Integration
- [ ] Selecting block in canvas shows PropertyPanel
- [ ] Changing params updates block.params
- [ ] Param change triggers patch recompilation
- [ ] Animation updates in real-time

### AC-5: Quality
- [ ] No TypeScript errors
- [ ] Component unit tests pass
- [ ] Works with existing domain blocks
